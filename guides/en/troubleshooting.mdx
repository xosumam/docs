title: Troubleshooting
description: Common upload, transcription, recorder, and TTS problems.
icon: life-ring
---

## File upload fails

Make sure the file is audio, not video. If the file is very large or the browser cannot read its duration, convert it to MP3 or M4A and try again.

## Not enough time

If the audio duration is longer than your balance, buy more transcription time from **Settings** or **Pricing**.

## Microphone does not start

Allow microphone permission in your browser. If you previously blocked it, open the browser site settings and enable microphone access for Xosum.am.

## Transcription takes too long

Long or noisy recordings can take longer to process. If the status does not change for a long time, refresh the page or check **History**.

## Transcription failed

Try uploading the file again. If the same file keeps failing, convert it to MP3 or M4A and retry.

## TTS does not generate audio

Shorten the text, change the tone, or remove sensitive content. Some text may not pass safety filters.

## Support

If the issue continues, email [support@xosum.am](mailto:support@xosum.am) and include your account email and recording link if available.
