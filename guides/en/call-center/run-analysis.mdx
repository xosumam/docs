title: Run AI analysis
description: Apply a QA checklist to one completed transcription.

---

AI analysis can only run after a recording has been transcribed.

<Info>
  Use single-recording analysis to review an important call manually or test a new checklist before applying it to a larger batch.
</Info>

## Run analysis

<Steps>
  <Step title="Open the recording details page">
    The recording must already be transcribed.
  </Step>
  <Step title="Go to Analysis">
    Existing analyses and the new-analysis action are shown here.
  </Step>
  <Step title="Choose a checklist">
    The system uses that checklist’s questions.
  </Step>
  <Step title="Start analysis">
    Results appear in the same section when processing is complete.
  </Step>
</Steps>

The analysis appears as a separate record and can move through these statuses: **Pending**, **Processing**, **Completed**, or **Failed**.

## Results

Click a completed analysis to open answers, scores, and comments for each question. You can also add a reviewer comment for internal QA notes.

## Delete analysis

If an analysis was run by mistake or is no longer needed, delete it from the recording’s **Analysis** tab.

<Note>
  Editing a checklist does not change the snapshot used by already completed analyses. New questions apply to future analyses.
</Note>
