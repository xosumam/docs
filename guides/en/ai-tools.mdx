The recording details page includes several AI tools for working faster with transcript content.

<Info>
  These tools are most useful for long recordings where you need quick answers, decisions, follow-up actions, customer issues, or meeting summaries.
</Info>

## AI chat

Click **Ask AI** and ask questions about the current transcript. Example prompts:

- “What was the main issue in this call?”
- “Write a follow-up email for the customer.”
- “Find the next steps mentioned in the conversation.”

The chat works within the context of the current recording. You can copy answers or start a new chat.

## Translation

In the **Translation** tab, choose a target language and click **Translate**. Supported options include English, Russian, French, German, Spanish, Persian, Arabic, Japanese, and Armenian.

## Summary

The **Analysis** tab may show a short AI-generated summary. You can send positive or corrective feedback if the summary is useful or inaccurate.

## Talk-time analysis

If speaker diarization succeeds, the **Analysis** tab includes speaker talk-time distribution. This is useful for interviews, meetings, and call-quality review.
