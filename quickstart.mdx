# üì° Xosum.am API Guide

Welcome to the **Xosum.am API documentation**. This API allows you to transcribe Armenian audio files by uploading them and receiving transcription results via webhook. It's a simple and powerful way to integrate transcription functionality into your platform.

---

## üîê Authentication

To use the API, you must:
- Be on the **Business** plan
- Have a valid **API key**
- Set a **webhook URL** and **webhook secret**

You can configure these in your account at [app.xosum.am](https://app.xosum.am).

---

## üõ†Ô∏è Endpoint

### `POST https://app.xosum.am/api/createTranscription`

This endpoint generates a signed upload URL and document ID.

#### Required Headers
- `Authorization: Bearer YOUR_API_KEY`

#### Required JSON Body

```json
{
  "type": "single_voice", // or "phone_call"
  "metadata": {
    "custom_key": "optional metadata"
  }
}
```

#### Example cURL Request

```bash
curl -X POST https://app.xosum.am/api/createTranscription \
  -H "Authorization: Bearer xosum_abcdef123456" \
  -H "Content-Type: application/json" \
  -d '{
        "type": "single_voice",
        "metadata": { "filename": "interview1" }
      }'
```

#### Sample Successful Response

```json
{
  "docId": "XYZ123",
  "uploadUrl": "https://storage.googleapis.com/..."
}
```

---

## üéß Uploading the Audio File

Once you receive the `uploadUrl`, use a PUT request to upload your **MP3** audio file.

#### Example using `curl`:

```bash
curl -X PUT -T ./my-audio.mp3 "https://storage.googleapis.com/..."
```

<Alert type="info" title="Note">
Upload must be an **MP3** file. Other formats like WAV/M4A are not supported via API.
</Alert>

## üíª Code Examples

### Python (Requests)

```python
import requests

api_key = "xosum_abcdef123456"
endpoint = "https://app.xosum.am/api/createTranscription"
headers = { "Authorization": f"Bearer {api_key}" }

payload = {
  "type": "single_voice",
  "metadata": { "filename": "my-audio.mp3" }
}

response = requests.post(endpoint, headers=headers, json=payload)
print(response.json())
```

### Node.js (Axios)

```javascript
const axios = require("axios");

const res = await axios.post(
  "https://app.xosum.am/api/createTranscription",
  {
    type: "single_voice",
    metadata: { filename: "my-audio.mp3" }
  },
  {
    headers: {
      Authorization: "Bearer xosum_abcdef123456"
    }
  }
);
console.log(res.data);
```
